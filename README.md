# AQM

## Get Project
```sh
git clone https://github.com/nemusou/AQM
# or Download ZIP from github
cd AQM
```

## Compile and Run
```sh
cd src
gcc example.c acf.c readfile.c findpeaks.c -o example
./example

# or

cd src
make
./example
```
