# AQM

## Get Project
```sh
git clone https://github.com/nemusou/AQM
# or Download ZIP from github
cd AQM
git checkout -b develop
git pull origin develop
```

## Compile and Run
```sh
cd src
gcc main.c acf.c readfile.c -o main.o
./main.o
```
